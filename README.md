# Plataforma de Dados de Risco de Crédito

Projeto para demonstrar uma pipeline completa de Data Engineering para análise de risco de crédito.

## Este projeto irá contemplar tecnologias como:

-Python
-Pandas
-DuckDB
-Docker
-Apache Airflow
-AWS S3
-Parquet
-Metabase
-Ollama
-SQL
